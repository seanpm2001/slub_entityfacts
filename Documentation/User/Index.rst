Users Manual
============

Target group: **Editors**

To use this plugin create a new content element on your page. You can now edit and configure the plugin in the backend.

To show any data in the frontend an ID of the Deutsche Nationalbibliothek (DNB) is required. There are several values available to be selected.
The order of the values/items in the frontend will be the same as in the backend. It is possible to select additional links to other sources like wikipedia, that is the last check mark.

With "/Resources/Public/Css/slub_entityfacts.css" it possible to alter the frontend view of this plugin. Further css files can be added in "/TypoScript/setup.ts".
